# Agence immobilière - Projet WEB (Clean architecture)

Le concept premier est de créer une copie d'un site typique d'agence immobilière sous le format d'une application web. Cependant nous allons utiliser la "Clean architecture" afin de mieux implémenter de nouvelles fonctionnalités et de nouvelles librairies facilement à notre projet.

## Cas d'utilisations

- Ajouter des produits
    - Doit contenir un ID Unique
    - Doit contenir un titre
    - Doit contenir un contenu
    - Peut contenir une image
    - Doit contenir un prix supérieur à 0€
- Supprimer un produit
- Modifier un produit
- Trouver un produit
